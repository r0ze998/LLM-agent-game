## 1. ゲーム概要

「村里」は、AIが自律的に行動するエージェントたちが暮らす世界を、プレイヤーが見守り・導くシミュレーションゲームです。

- **128×128マス**の手続き生成ワールド（平原・森林・山岳・水域・砂漠・湿地）
- 最大**100体のAIエージェント**が自律的に生活・会話・繁殖
- エージェントが集まると**村が自然発生**し、政治・文化・経済が芽生える
- **4X戦略**（探索・拡張・開発・殲滅）で村を発展させる
- エージェント同士の**会話はLLMがリアルタイム生成**

### ゲームモード

| モード | 説明 |
|--------|------|
| **プレイヤーモード** | 魂の描写から自分のエージェントを作成し、村を率いて勝利を目指す |
| **観察モード** | 世界の自律的な発展を見守る。介入は「天の声」のみ |

## 2. 始め方

### プレイヤーモードで開始

1. タイトル画面で **「始める」** をクリック
2. **名前**を入力（省略するとAIが自動命名）
3. **魂の描写**を10文字以上で記述
   - 例：*「穏やかな農夫で、土地と季節のリズムを深く敬う。争いを嫌い、常に対話で解決しようとする。」*
   - この記述からAIが性格・哲学・スキルを自動決定する
4. **「世界に降り立つ」** をクリック

### 観察モードで開始

1. タイトル画面で **「観察する」** をクリック
2. 8体のエージェントが生成された世界が即座に開始

### ヘッドレスモードへの自動接続

サーバーが `HEADLESS=true` で起動している場合、タイトル画面を開くと自動的にそのゲームに観察者として接続します。

## 4. エージェント（住人）

### 基本属性

エージェントは以下の属性を持ちます:

**アイデンティティ**
- 名前、世代（0が初代）、年齢、状態（子供/成人/老人/死亡）

**性格（5軸、0〜100）**

| 軸 | 高い場合 | 低い場合 |
|----|---------|---------|
| 開放性 | 探求的、好奇心旺盛 | 保守的、慎重 |
| 協調性 | 親切、協力的 | 競争的、独立的 |
| 誠実性 | 勤勉、計画的 | 自由奔放、即興的 |
| 勇敢さ | 大胆、リスク許容 | 慎重、安全志向 |
| 野心 | 権力志向、目標達成型 | 穏やか、現状維持型 |

**哲学**
- 統治思想: 民主主義 / 能力主義 / 権威主義 / 無政府主義 / 神政政治
- 経済思想: 集団主義 / 市場経済 / 贈与経済 / 封建制
- 価値観: 3つの価値（勇気・知恵・慈悲・正義・自由・調和・名誉・繁栄 等）
- 世界観: 自由テキスト

**スキル（8種、0〜100）**

| スキル | 効果 |
|--------|------|
| 農業 (Farming) | 食糧生産効率 |
| 建築 (Building) | 建設速度 |
| 工芸 (Crafting) | アイテム作成 |
| 統率 (Leadership) | 選挙・統治 |
| 戦闘 (Combat) | 軍事行動 |
| 外交 (Diplomacy) | 交渉・関係構築 |
| 教育 (Teaching) | 子供の成長促進 |
| 治療 (Healing) | 回復・医療 |

### 3つのニーズ

エージェントは毎tick、ニーズが減衰します。20以下になると**危機状態**となり、本能的に対処行動を取ります。

| ニーズ | 減衰/tick | 回復手段 | 回復量 |
|--------|----------|---------|--------|
| 空腹 | -1 | 食事 | +30 |
| 体力 | -2 | 睡眠 | +50 |
| 社交 | -0.5 | 交流 | +20 |

### 行動の決定

エージェントは3段階の優先度で行動を決めます:

1. **本能（P2）** — 空腹20以下 → 食べる/採集、体力20以下 → 睡眠
2. **日課（P1）** — LLMが生成する1日24時間の行動計画
3. **状況判断（P0）** — 計画にない状況でのLLM即時判断

LLM未接続時は、ルールベースの代替計画が使われます:
- 朝（6〜12時）: 最も得意なスキルに基づく作業
- 昼（13〜17時）: 探索・採集
- 夕（17〜19時）: 社交
- 夜（19〜5時）: 睡眠
- 食事時間（5時・12時・19時）: 食事/採集

### ライフサイクル

```
誕生 → 子供期(0〜200tick) → 成人期(200〜寿命75%) → 老年期(〜寿命) → 死亡
```

**誕生条件:**
- 両親が成人エージェント
- 互いの好感度が +50 以上
- 食糧余剰が 50 以上
- 毎tick 2% の確率

**子供:**
- 性格は両親の平均 ± ランダム変異
- スキルは両親から継承（ブレンド）
- 名前は両親の名前から生成
- 教師役の大人から学べる

**老人:**
- 上位3つの長期記憶が「村の知恵」として蓄積
- 若い世代の行動計画に影響を与える

### 記憶システム

エージェントは3層の記憶を持ちます:

| 層 | 容量 | 用途 |
|----|------|------|
| 作業記憶 | 20件 | 直近の重要な出来事 |
| エピソード記憶 | 200件 | 過去の体験・会話 |
| 長期記憶 | 無制限 | 内省から得た知恵・信念 |

記憶のスコアリング: `スコア = 新しさ × 重要度 × log(参照回数 + 2)`

### 関係性

エージェント間の関係は3つの数値で管理されます:

| 指標 | 範囲 | 説明 |
|------|------|------|
| 好感度 (Sentiment) | -100〜+100 | 感情的な好悪 |
| 信頼度 (Trust) | 0〜100 | 信用の程度 |
| 親密度 (Familiarity) | 0〜100 | 付き合いの深さ |

- 関係は時間経過で緩やかに減衰（-0.5/tick）
- 会話・協力で強化される
- 好感度 +50 以上で子供が生まれうる
- 好感度 -30 以下で「ライバル」関係

### 会話

8マス以内にいるエージェント同士が会話します。

**優先度（高い順）:**
1. 初対面（親密度 < 5）: 60%
2. 久しぶり（50+ tick以上）: 50%
3. ライバル関係: 45%
4. 親友: 40%
5. 一般的な遭遇: 30%

1 tickあたり最大3組が同時に会話できます。

**会話の結果:**
- 好感度・信頼度・親密度の変化
- 新しい記憶の生成
- 情報交換（知識の伝播）
- 文化交流（異なる村のエージェント間）

## 5. 村の運営

### 村の誕生

村は自然発生します:
- **3人以上のエージェント**が近くに集まる
- **20 tick以上**共存する
- 合議で**村が創設**される

### 統治タイプ

| タイプ | 指導者の選び方 | 特徴 |
|--------|-------------|------|
| 民主主義 | 全員投票 | 480 tickごとに選挙 |
| 能力主義 | スキル最高者 | 統率スキルが基準 |
| 権威主義 | 任命制 | 指導者が後継を指名 |
| 無政府主義 | 不在 | 全員が対等 |
| 神政政治 | 宗教指導者 | 信仰を基盤 |

**統治ドリフト**: 村の成人の60%以上が同じ統治思想を持つと、村の統治タイプがその方向に変化します。

### 法律

村は法律を制定できます:
- 指導者が法案を提案
- 議会メンバーが投票
- 可決された法律は村全体に影響

### 文化

各村は独自の文化を持ちます:
- **伝統**: 行動規範
- **物語**: 歴史的な語り
- **タブー**: 禁忌事項
- **命名法**: 子供の命名規則
- **挨拶**: 社交の様式
- **建築様式**: 建物の外観

### 宗教

特定条件で宗教が自然発生します:
- 老人エージェントの哲学から生まれる
- 信者はエージェント間の会話で増える
- 正統性（0〜100）が村の結束に影響
- 文化ポイントにボーナス

## 6. 資源と経済

### 5つの資源

| 資源 | 主な入手方法 | 用途 |
|------|-------------|------|
| 食糧 (Food) | 農場、採集 | 人口維持（0.5/人/tick）、訓練 |
| 木材 (Wood) | 伐採、森林タイル | 建設 |
| 石材 (Stone) | 採掘、山岳タイル | 建設、防壁 |
| 鉄 (Iron) | 採掘 | 軍事ユニット、高度な建物 |
| 金 (Gold) | 採掘、交易 | 高度な研究、交易 |

### 経済の仕組み

- エージェントが採集した資源の **50%が村に課税**される
- 残り50%はエージェントの個人在庫
- 村の資源は建設・研究・訓練に使用される
- 食糧が0になると**飢餓**が発生し人口が減少（-5%/tick）

### 環境劣化

- 採集は土地の劣化度を +0.02 上昇させる
- 自然回復: +0.001/tick
- 劣化度80%超で**荒廃地**になる
- 過剰な農業で肥沃度が低下

### 交易

村同士で交易が可能です:
- 交易路を確立（距離が効率に影響: 1マスあたり5%コスト）
- 道路建設で効率+2%（最大+30%）
- 自動的に資源交換が行われる

## 9. 軍事と戦闘

### ユニット一覧

| ユニット | 攻撃 | 防御 | 速度 | 解放条件 |
|---------|------|------|------|---------|
| 民兵 (Militia) | 低 | 低 | 普通 | 初期 |
| 戦士 (Warrior) | 中 | 中 | 普通 | 兵舎 |
| 弓兵 (Archer) | 中 | 低 | 普通 | 射撃場 |
| 槍兵 (Spearman) | 中 | 高 | 普通 | 兵舎 |
| 騎兵 (Cavalry) | 高 | 中 | 速 | 厩舎 |
| 攻城槌 (Siege Ram) | 高 | 低 | 遅 | 攻城工房 |
| 投石機 (Catapult) | 最高 | 最低 | 遅 | 攻城工房 |
| 騎士 (Knight) | 最高 | 高 | 速 | 厩舎+鋼鉄 |
| 銃兵 (Musketeer) | 最高 | 中 | 普通 | 火薬 |
| 精鋭衛兵 (Elite Guard) | 最高 | 最高 | 普通 | 軍事完全習得 |

### 戦闘の仕組み

1. **宣戦布告** → 外交状態が「戦争」に変更
2. **軍の移動** → 目標に向けてパスファインディング
3. **戦闘解決**:
   - 攻撃力 = Σ(ユニット攻撃 × 数 × 練度) × ランダム(0.8〜1.2)
   - 防御力 = Σ(ユニット防御 × 数 × 練度) + 駐屯ボーナス
   - **勝者**: 合計パワーが高い方
   - **損耗**: 攻撃側 40%、防御側 30%
   - **練度**: 戦闘ごとに +10（最大100）

## 10. 外交

### 外交状態

| 状態 | 効果 |
|------|------|
| 友好 (Friendly) | 交易ボーナス |
| 中立 (Neutral) | 標準 |
| 敵対 (Hostile) | 緊張度上昇 |
| 同盟 (Allied) | 共同防衛・交易ブースト |
| 戦争 (War) | 軍事攻撃可能 |

### 緊張度

- 0〜100のスケール
- 領土の重複で +5/tick 上昇
- 交易で緩和
- 高い緊張度は戦争のリスクを高める

### 外交コマンド

| コマンド | 効果 |
|---------|------|
| 同盟提案 | 相互防衛条約を締結 |
| 宣戦布告 | 戦争状態に移行 |
| 和平提案 | 戦争を終結 |
| 同盟破棄 | 同盟を解消 |

## 11. 天の声（意図システム）

プレイヤーは「天の声」としてエージェントに間接的な指示を送れます。直接操作ではなく、**示唆・導き**として機能します。

### 意図の種類

| タイプ | 説明 | 例 |
|--------|------|-----|
| 導き (Guide) | 行動を提案 | 「食糧をもっと集めよう」 |
| 価値 (Value) | 哲学を強化 | 「勇気を何より重んじよ」 |
| 警告 (Warning) | 危険を通知 | 「東の侵略者に注意せよ」 |
| 問い (Question) | 倫理的問題を提起 | 「拡張は犠牲に値するか？」 |
| 命名 (Name) | 名前を提案 | 「この子を"ヒカリ"と名付けよ」 |
| 関係 (Relationship) | 関係性を促進 | 「アキラと協力せよ」 |

### 強さ

| 強さ | 効果 |
|------|------|
| 囁き (Whisper) | 微かな影響、抵抗されやすい |
| 提案 (Suggestion) | 明確な方向性 |
| 勅令 (Decree) | 強い命令、ほぼ従う |

**注意**: エージェントの性格に反する指示は抵抗される場合があります。

### 対象

- **エージェント**: 特定の1人に送る
- **村**: 村の全メンバーに送る
- **世界**: 全エージェントに送る

## 12. 自治システム

### 規約（Covenant）

村が制定できるルール・契約です。

**条項の例:**
- 税率の設定（0〜50%）
- 交易関税（0〜30%）
- 徴兵制（人口の0〜20%）
- 資源共有（0〜50%）
- 建物禁止/補助
- 研究集中（特定分野+10〜50%）
- 軍事同盟
- 不可侵条約
- 移民政策（開放/閉鎖）
- 配給制（食糧消費50〜100%）
- 祭典（文化ボーナス、食糧コスト）

**適用範囲:**
- 村内: 提案した村のみ
- 二者間: 2つの村の間
- 全体: 全村に適用

### 発明（Invention）

村が新しい建物・技術・ユニットを発見するシステムです。
- 発明者エージェントが作成
- 他の村にはエージェントの移動・会話で伝播
- 時間経過で関連度が低下

### 機関（Institution）

村が設立できる組織です。

| タイプ | 役割 |
|--------|------|
| ギルド | 経済・工芸の専門集団 |
| 宗教 | 信仰の共同体 |
| 同盟 | 軍事協力 |
| 学院 | 知識共有 |

- 入会条件あり（人口・技術・文化など）
- メンバーにステータスボーナス
- 共同資金（Treasury）を持つ

## 13. 勝利条件

5つの勝利条件があります。いずれかを最初に達成した村が勝利します。

| 勝利タイプ | 条件 | 攻略のポイント |
|-----------|------|-------------|
| **スコア勝利** | tick 12,000時点で最高スコア | バランスよく発展 |
| **制覇勝利** | 全村の75%を支配 | 軍事力を集中投資 |
| **文化勝利** | 文化ポイント1,000達成 | 神殿・劇場・記念碑を建設 |
| **外交勝利** | 全村の60%と同盟 | 交易と外交を重視 |
| **技術勝利** | 3分野すべて完全習得 | 研究施設と図書館を最優先 |

**スコアの計算要素:** 人口 + 技術数 + 文化ポイント + 軍事力

## 15. Tips & 戦略ガイド

### 序盤（1〜50 tick）

1. **食糧を最優先** — 農場を2〜3棟建てて飢餓を防ぐ
2. **家を建てる** — 住居容量を確保しないと人口が伸びない
3. **技術: 農業 → 灌漑** — 食糧生産の基盤を固める
4. **探索させる** — エージェントに周囲の資源を発見させる

### 中盤（50〜200 tick）

5. **研究施設** — 図書館・学院で技術開発を加速
6. **交易路** — 隣村と交易して不足資源を補う
7. **外交** — 同盟を結んで安全を確保
8. **勝利条件を絞る** — 全方位より1〜2の勝利パスに集中

### 終盤（200 tick〜）

9. **軍事バランス** — 制覇勝利を狙わなくても最低限の防衛力は必要
10. **文化投資** — 神殿・劇場で文化ポイントを稼ぐ
11. **規約の活用** — 研究集中や軍事同盟で村を強化
12. **災害対策** — 倉庫で備蓄を増やし、災害に備える

### プレイスタイル別

**平和的発展型**
- 文化勝利 or 技術勝利を目指す
- 交易と同盟を重視
- 学校・図書館・神殿を多く建てる

**軍事征服型**
- 制覇勝利を目指す
- 兵舎・鍛冶場・攻城工房を優先
- 軍事技術ツリーを急ぐ

**外交型**
- 外交勝利を目指す
- 交易で関係を強化
- 規約で不可侵・同盟を結ぶ
- 集会場・市場を建てる

### 天の声の効果的な使い方

- **序盤**: 「食糧を集めよう」で生存を安定させる
- **中盤**: 「この村と協力せよ」で同盟を促進
- **危機時**: 「東に敵が迫っている」で防衛意識を高める
- **文化**: 「知恵を重んじよ」で研究志向を強化
